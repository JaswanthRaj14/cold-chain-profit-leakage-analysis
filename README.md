# Cold Chain Profit Leakage Analysis

## Overview
This project analyzes profit leakage in a cold-chain logistics business scenario. Despite expansion across multiple regions, the company experienced declining profitability driven by rising waste costs—especially in the pharmaceutical category. The objective was to identify the root causes of waste and profit loss using data analytics and build an interactive dashboard for decision-making.

---

## Problem Statement
FreshGuard Solutions, a cold-chain logistics company handling temperature-sensitive products, observed significant profit leakage.  
Internal teams proposed different explanations:
- Supply chain team suspected temperature deviations
- Procurement team blamed supplier quality
- Retail team attributed losses to discount strategies

The goal was to use data to identify the true drivers of waste and profitability issues.

---

## Approach
1. Data Cleaning and Validation using Python  
2. Exploratory and Diagnostic Analysis using PostgreSQL  
3. Hypothesis Testing across operational, supplier, pricing, and inventory factors  
4. Data Visualization using Power BI  
5. Dashboard development to communicate insights  

---

## Tools Used
- Python (Pandas) – Data cleaning and preprocessing  
- PostgreSQL – Exploratory and diagnostic SQL analysis  
- Excel – Initial data validation and inspection  
- Power BI – Interactive dashboard visualization  

---

## Key Business Questions
- Which product categories contribute most to waste?
- Are temperature deviations causing spoilage?
- Does distribution time affect waste cost?
- Do suppliers influence product spoilage?
- Are discount strategies reducing profitability?
- Do promotions reduce waste?

---

## Key Insights
- Waste cost accounted for a significant portion of total revenue
- Pharmaceuticals contributed the majority of waste costs
- Longer distribution times increased spoilage risk
- Supplier quality had minimal impact on waste
- Heavy discounting resulted in negative profit margins
- Highly sensitive products amplified operational losses

---

## Dashboard Features
- Executive KPI summary (Revenue, Waste Cost, Profit, Margin)
- Waste Cost by Product Category
- Regional Profit Leakage Analysis
- Discount vs Profit Margin Trend
- Distribution Hours vs Waste Cost
